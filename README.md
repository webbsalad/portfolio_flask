# Веб-сайт портфолио

Это веб-сайт портфолио, созданный с использованием Flask. Сайт предназначен для сдачи работ.

## Функционал

- **Главная страница**
- **Информация о себе**
- **Ссылки на контакты**
- **Страницы проектов**

## Структура проекта:
```
D:.
├───static               // статистические файлы
│   ├───css
│   └───img
├───templates            // html файлы
├───translations         // файлы для перевода стираницы
│   ├───en
│   │   └───LC_MESSAGES
│   └───ru
│       └───LC_MESSAGES
└───app.py               // основной файл
```
